# Exhibitions 詳細ページ ACF フィールド定義

## 実装済みセクション（Section 01〜07）

| セクション | フィールド名            | 型               | 説明                       |
|:-----------|:------------------------|:-----------------|:---------------------------|
| 01 MV      | `main_visual`           | グループ         | メインビジュアル           |
|            | → `image`               | 画像             | MV画像                     |
|            | → `text1`               | テキストエリア   | タイトルテキスト           |
|            | → `text2`               | テキスト         | 会期テキスト               |
| 02 リリース | `release`              | グループ         | リリース資料               |
|            | → `filename1` / `url1`  | テキスト / URL   | 資料1（最大3組）           |
|            | → `filename2` / `url2`  | テキスト / URL   | 資料2                      |
|            | → `filename3` / `url3`  | テキスト / URL   | 資料3                      |
| 02 連絡先  | `contact`               | 真偽値           | Contactボタン表示          |
| 04 説明    | `description`           | テキストエリア   | 展示会説明文               |
| 05 概要    | `overview`              | WYSIWYG          | 展示概要（HTML）           |
|            | `overview_note`         | テキストエリア   | 概要の補足                 |
| 06 声明    | `statement`             | テキストエリア   | ステートメント             |
|            | `contribution`          | テキストエリア   | 寄稿文                     |
| 07 作品    | `featured_works`        | リレーション     | Artworks投稿を選択         |
|            | `featured_works_column` | セレクト         | カラム数（1/2/3）          |

## 未実装セクション（Section 08〜15）

各セクションのACFフィールド名を埋めてください。

### Section 08: CONTACT CTA

セクション間のContactボタン。
contact_url未設定時は非表示。

| フィールド名 | 型         | 説明                         | 必須 |
|:-------------|:-----------|:-----------------------------|:-----|
| `contact_url`| URL        | ContactボタンのURL(テキストはCONTACTのまま)           |      |

### Section 09: EDITION

展示に関連するEditions投稿の一覧。

| フィールド名       | 型           | 説明                                       | 必須 |
|:-------------------|:-------------|:-------------------------------------------|:-----|
| `editions`         | リレーション | Editions投稿を選択（editions投稿タイプ）   |      |

※ 各Editionの画像・artist_name・title は、選択先投稿のカスタムフィールドから取得。

### Section 10: BOOKS

展示に関連するBooks投稿の一覧。

| フィールド名 | 型           | 説明                                 | 必須 |
|:-------------|:-------------|:-------------------------------------|:-----|
| `books`      | リレーション | Books投稿を選択（books投稿タイプ）   |      |

※ 各Bookの画像・artist_name・title は、選択先投稿のカスタムフィールドから取得。

### Section 11: EVENTS

展示に関連するイベント情報。

| フィールド名           | 型             | 説明               | 必須 |
|:-----------------------|:---------------|:-------------------|:-----|
| `events`               | リッチエディター | wysiwyg       |      |

### Section 12: ARTISTS

展示参加アーティストの一覧。

| フィールド名 | 型           | 説明                                     | 必須 |
|:-------------|:-------------|:-----------------------------------------|:-----|
| `artists`    | リレーション | Artists投稿を選択（artists投稿タイプ）   |      |

※ 各Artistの画像は `mv_images->pc`、名前は投稿タイトル、プロフィールは `description` フィールドから取得。

### Section 13: INSTALLATION VIEWS

展示風景のスライダー画像。

| フィールド名           | 型         | 説明                 | 必須 |
|:-----------------------|:-----------|:---------------------|:-----|
| `installation_views`   | リピーター | 展示風景画像         |      |
| → `item`              | 画像       | 画像                 |      |

### Section 14: FILM

展示関連の映像。

| フィールド名       | 型             | 説明                       | 必須 |
|:-------------------|:---------------|:---------------------------|:-----|
| `film`             | グループ       | Film設定                   |      |
| → `youtube_url`    | URL            | YouTube動画URL             |      |
| → `caption`        | テキスト       | 動画キャプション           |      |
| → `see_more_on_youtube_url`    | URL            | YouTubeチャンネルリンク    |      |
| → `event_title`    | テキスト       | 記事タイトル               |      |
| → `event_description` | テキストエリア | 記事本文                   |      |

### Section 15: PRODUCT

展示関連プロダクト。
shop_name, shop_url は descriptionの下に `shop_name` : `shop_url` の表記でリンクを表示する。

| フィールド名   | 型             | 説明             | 必須 |
|:---------------|:---------------|:-----------------|:-----|
| `product`      | グループ       | Product設定      |      |
| → `image`      | 画像           | プロダクト画像   |      |
| → `description`       | テキストエリア | 説明テキスト     |      |
| → `shop_name` | テキスト | ショップ名 |  |
| → `shop_url`  | URL     | ショップURL | |

---

## 備考

- リレーションフィールド（`editions`, `books`, `artists`）は、Section 07 の `featured_works` と同じパターン。選択先投稿の既存カスタムフィールドからデータを取得する。
- 各セクションはフィールドが空の場合、セクション自体を非表示にする。
- Installation Views は既存の Swiper スライダーを使用。
